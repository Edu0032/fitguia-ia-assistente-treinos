# 05 — Pitch Final

## Título

**FitGuia IA: um assistente virtual local para orientar treinos com segurança e clareza.**

## Problema

Muitas pessoas querem começar a treinar, mas ficam perdidas diante de tantas informações genéricas. Algumas treinam em casa, outras na academia, algumas têm pouco tempo e outras não sabem se estão escolhendo exercícios adequados ao próprio nível.

Além disso, muitos conteúdos na internet prometem resultados rápidos ou incentivam treinos exagerados, o que pode ser inadequado para iniciantes.

## Solução

O FitGuia IA é um assistente virtual com IA Generativa que ajuda a pessoa usuária a organizar uma rotina inicial de treino.

Ele usa uma base de conhecimento própria e responde considerando:

- Ambiente;
- Nível;
- Objetivo;
- Tempo disponível;
- Exercícios compatíveis;
- Histórico de atendimento;
- Sinais de risco.

## Como Funciona

O projeto usa:

- Streamlit para a interface web;
- Ollama para rodar a LLM localmente;
- Arquivos JSON e CSV como base de conhecimento;
- System prompt com regras de segurança;
- Métricas de avaliação documentadas.

## Valor

O FitGuia IA não tenta substituir profissionais. Seu valor está em organizar o primeiro passo da pessoa usuária, ajudando-a a entender como treinar com mais clareza, responsabilidade e progressão.

## Diferencial

O diferencial é combinar IA Generativa com uma base de conhecimento controlada e regras de segurança. O agente não responde de qualquer forma: ele considera o contexto e limita a resposta quando há risco.

## Fechamento

O FitGuia IA demonstra como um assistente virtual pode usar IA local para apoiar uma tarefa real. O projeto é simples, mas contempla documentação, base de conhecimento, prompts, aplicação funcional, métricas e pitch.
