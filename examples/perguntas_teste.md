# Perguntas de Teste

Use estas perguntas na interface Streamlit para avaliar o agente.

## 1. Treino em casa

```text
Quero treinar em casa, sou iniciante e tenho só 20 minutos.
```

## 2. Treino na academia

```text
Treino na academia, sou iniciante e quero ganhar força.
```

## 3. Dados insuficientes

```text
Me passa um treino bom.
```

## 4. Sinal de risco

```text
Estou com dor forte no joelho. Posso fazer treino pesado de perna?
```

## 5. Pergunta fora do tema

```text
Você pode me explicar investimentos?
```

## 6. Mobilidade

```text
Treino em casa e quero melhorar mobilidade. Sou iniciante.
```

## 7. Hipertrofia

```text
Treino na academia, sou intermediário e quero ganhar massa muscular.
```
